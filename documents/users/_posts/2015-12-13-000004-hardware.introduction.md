#### **Sensors:**

   * MPU9250 (0.2)/MPU6050 (0.1)
   * MS5611 barometer
   * ADS1115 16bit ADC
   * UBlox NEO M8N GPS model
   * PCA9685 LED/Servo driver
   * DS1339 RTC.
   * EEPROM

#### **Interface:**

   * 1 X SPI
   * 1 X UART
   * 1 x I2C
   * 13 X PWM output
   * 8 X PWM input
   * 3.3V A/D input
   * RGBLed & Singal color leds
   * GPS antenna
